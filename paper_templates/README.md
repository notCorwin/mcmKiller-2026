# 数模论文行文思路与格式规范

- 论文使用`LuaLaTeX`编译；
- 论文题目为**三号黑体字**；
- 一级标题为**四号黑体字**，**居中**；
- 二级、三级标题为**小四号黑体字**，**左端对齐**（不居中）；
- 其他一律采用**小四号宋体字**；
- **单倍行距**；
- 论文中**不能**有页眉。

## 标题、摘要和关键词｜Title, Abstract and Keywords

依照以往习惯，这三个部分整体要控制在PDF的一页之内。

### 摘要

**行文思路**：每段用“**针对问题X，**”开头，说明针对该问题建立了什么模型；然后按照建模的步骤**逐层递进**，描述**核心方法**和**数学工具**；最后给出具体的数值结果并收尾。

#### 格式

- **加粗**：“针对问题X，”、重要的模型、算法和思路。

## 问题重述｜Restatement

用**自己的话**，把题目背景和每个小问的核心需求重新说一遍，**简短**、清晰、**可读**，为后续“问题分析”和“建模”铺路。

## 模型假设｜Assumptions

为了能够建模，对现实问题做的所有简化和约定。

### 格式

- 有序列表，按条列出。

## 符号说明｜Notations

只列**全篇常用或重要**的符号，临时的变量不用放。

正文中首次出现某个符号时，也要**括号说明一次**（如“令 $r_i$ 为第 $i$ 个定日镜半径（见表 1）”）。

### 格式

1. **斜体**表示变量，**正体**表示单位。
2. 符号**单字母**优先，用**上/下角标**区分。

## 问题分析｜Analysis

首先抓住题目的本质，将现实问题抽象成数学和物理问题；然后梳理解题思路，建立模型框架；最后描述求解策略。

### 格式

- 每个问题单独一个二级标题。
- 避免结论性描述，只陈述分析思路。

## 模型建立与求解｜Modeling

### 模型的建立

1. **第一段**一般直接引用”问题分析“部分的内容，并在下方附一张模型建立的**流程图**。
2. 一个问题的模型通常由多个子模型整合而成，每个子模型又由多个方程组成。因此此节通常采用递进逻辑叙述：模型的建立->子模型的建立->方程的建立。每个子模型的建立通常采用递推逻辑叙述。

### 模型的求解

此节包括使用的**算法**、**程序流程设计**和**结果展示与结果分析**。

1. 对于结果展示部分的**仿真图**，需要在图片左/右侧单独添加文本框，解读仿真图体现的信息。

## 模型评价｜Evaluation

分析论文中使用模型的**优点**、（**不致命**的）**缺点**和**推广应用**。

### 格式

1. 二级标题用于：优点、缺点、推广。
2. 每节使用有序列表分析3～5点。

## 参考文献｜References

### 人工智能工具使用情况

使用AI工具生成的内容，应在正文相应位置进行标注；在参考文献中列出所用AI工具，格式和示例如下： 

```
[编号] 工具名称，版本/型号，开发机构/公司，使用日期。
[1] DeepSeek，DeepSeek-V4, 深度求索（DeepSeek）, 2026-04-02。
```

## 附录｜Attachments

### 代码

展示完整的问题求解代码。

### AI工具使用详情

- 使用AI工具名称和版本
- 具体使用目的和环节
- 关键交互记录（重要提示词和回复）
- 采纳和人工修改情况

## 杂项｜Nontrivial

### TeX Package Recommendation

#### 中文

- **ctex**：提供中文排版的核心支持。

#### 数学和物理公式

- **amssymb**：基于amsfonts，提供大量额外数学符号。
- **bm**：输出粗体数学符号。
- **mathtools**：amsmath的增强版，用于数学公式排版。
- **siunitx**：统一规范的数字和 SI 单位排版方案。

#### 交叉引用

- **cleveref**：自动根据引用对象添加正确前缀（图1、表1）；写中文论文时需要重定义英文前缀。

#### 表格

- **tabulary**：按每列实际内容量分配宽度。
  - **adjustbox**：在tabulary的基础上还支持设置宽和高，超出范围自动缩放。
- **booktabs**：三线表。三线分别为toprule，midrule和bottomrule。
- **multirow**：跨行或跨列合并表格。

#### 代码

- **minted**：代码排版。

#### 图表生成

- **pgfplots**：基于TikZ。可以为流程图自动布局，全自动处理复杂绘图。
  - 3D 绘图：曲面图、网格、等高线；
  - 统计图：柱状图、箱线图、直方图、堆积图；
  - 数据处理：外部表格、坐标变换、颜色映射；
  - 极坐标和特殊图：Smith、三元图、箭头场；

#### 页面布局

- **geometry**：精确控制页边距。
- **fancyhdr**：自定义页眉页脚。
